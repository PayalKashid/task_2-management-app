# Task Management Application

## Tech Stack

### Backend
- FastAPI
- MongoDB
- JWT Authentication
- Pydantic
- Uvicorn

### Frontend
- React
- TypeScript
- Tailwind CSS
- Axios

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication

### Projects
- Create Project
- View Projects
- Delete Projects

### Tasks
- Create Tasks
- View Tasks
- Delete Tasks
- Filter by Status
- Filter by Priority

## Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
